# book-rest-api
A simple REST API built with Spring Boot demonstrating CRUD operations.

## Technologies
- Java
- Spring Boot
- Maven

## Features

- Create a book
- View all books
- View a book by ID
- Update a book
- Delete a book

## Endpoints
| Method | Endpoint |
|--------|----------|
| POST | /books |
| GET | /books |
| GET | /books/{id} |
| PUT | /books/{id} |
| DELETE | /books/{id} |
